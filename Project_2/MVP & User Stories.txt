MVP (Minimum Valuable Product)
1. Allow users access to selected a Twitter account
   1. Use Twitter API to retrieve a Twitter user’s timeline, for example, posts
2. Analyzing the sentiment of posts by using Google API
User Stories
1. Reporters, social media researchers, or students
   1. A group of people who want to analyze the sentiment of the tweets of a politician or a celebrity 
     Modular design
1. User input 
   1. A politician or a celebrity’s Twitter account
   2. Set the number of tweets that one wants to retrieve
   3. Retweets, replies, and original tweets
2. Twitter API
   1. Retrieve tweets base on user’s input
3. Google API
   1. Remove blocks and embeddings
   2. Retrieve a sentiment analysis result from Google
4. Results
   1. Show the result to the user
